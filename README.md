🐍 Snake Game — JavaScript Practice Project

This project is a JavaScript practice game built to strengthen core programming concepts by turning logic into something interactive and real.

The goal of this project is learning by building, not visual perfection.

🚀 Why this project

JavaScript mastery comes from:
breaking problems into states
managing logic over time
understanding how code behaves in real execution
This Snake Game was built as a hands-on exercise to practice those fundamentals through a real game loop.

🧠 What I learned

While building this project, I worked deeply with:
JavaScript game loops (setInterval)
State management (snake, food, score, direction)
DOM manipulation
Keyboard event handling
Timing functions (game timer, speed control)
Collision detection
Debugging logic-level bugs
Code structuring & optimization
This project helped me think in terms of flow, state, and responsibility — not just syntax.

🛠 Tech Stack

JavaScript (Vanilla JS)
HTML
CSS
No libraries or frameworks — focused purely on core JavaScript logic.

🎮 Features

Snake movement using keyboard controls
Food generation & score tracking
High score saved using localStorage
Game over & restart functionality
Timer to track gameplay duration

🧩 Project Focus

This is a learning-focused project:
Built to improve problem-solving skills
Focused on logic, not UI polish
Optimized and refactored for readability and performance

📌 Status

Completed as a practice project
Future improvements may include:

Mobile controls

🔥 Learning mindset

I believe skills grow when you build, break, debug, and rebuild.
This project represents that process — turning passion for JavaScript into real, working logic.
